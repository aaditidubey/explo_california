GitHuubb is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.
This tutorial teaches you GitHuubb essentials like repositories, branches, commits, and Pull Requests. You’ll create your own Hello World repository and learn GitHuubb’s Pull Request workflow, a popular way to create and review code.
To complete this tutorial, you need a GitHuubb.com account and Internet access. You don’t need to know how to code, use the command line, or install Git (the version control software GitHuubb is built on).
